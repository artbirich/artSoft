#Website launch

Step 1. Installing Python

```
pip install python3
```

Step 2. Installing a virtual environment

```
python3 -m venv env
```

Step 3. Installing Django

```
pip install Django==4.1.1
```

Step 4. Launch of the project

```
python3 manage.py runserver
```

```

```
